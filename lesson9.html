"Hay" in English

In order to say "Hay" in English, we need to establish if what we are talking about is singular or plural, if we are talking about just one thing we use "there is"

There is a problem - Hay un problema
There is a car outside - Hay un coche afuera

When we are talking about multiple things we use "there are" which also translates to "hay"

There are problems - Hay problemas
There are cars outside - Hay coches afuera

When we are talking about the past we wil also use one of the two versions depending on whether it is plural or singular. If it is simgular then we should use "there was"

There was a problem - Hubo un problema
There was a car outside - Había un coche afuera

When we are talking about multiple things we use "there were"

There were problems - Había problemas
There were cars outside - Había coches afuera

Finally when we use this structure in the Future we use "There will be" both for plural and singular

There will be problems - Habrá problemas
There will be a car outside - Habrá un coche afuera

As you can see this structure is also build on the basic "to be" structure

Reflexive pronouns

In this section we'll keep exploring pronouns. When we want to emphasize that we did something or will do something all by ourselves we would use these pronouns:

Myself - Yo mismo
Yourself - Tu mismo
Ouselves - Nosotros mismos
Themselves - Ellos mismos
Himself - Él mismo
Herself - Ella mismo

Notice how in the words "Ourselves" and "Themselves" there is "ves" instead of "f". It's because these words are used in plural so they are pronounced in plural as well. The word "Yourself" can also be used in plural as "Yourselves" if you're refering to multiple people

The usage of these pronouns could be tricky though. Basically any time we have to say something like this: "I like me" or "John looks at John" we will need these kind of pronouns. Instead we will say "I like myself" and "John looks at himself".


Istead of "Jane cooks for Jane" or "Jane cooks for her" we would use "Jane cooks for herself"
